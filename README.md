# Dev-Connector
A MERN Stack Social Network Website for Developers. 

## Features

- Social Media App
- User can register and Login
- Add Education
- Add Experience
- Add Github profile thorough API
- Post something
- View Posts
- Like,Dislike,Comment on Posts
